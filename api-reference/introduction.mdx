PulsejetDB provides REST API endpoints for testing and trial purposes. REST API isn't meant for performance oriented use cases.

This section introduces REST API endpoints provided by PulsejetDB to operate on vectors.

Follow these steps to get started:

<Steps>
  <Step title="Set up the development environment">
    Learn more about [usage here](/rest/usage). 
  </Step>
  <Step title="Connect to a local or remote instance.">
    Connect your data source to MindsDB via [this endpoint](/rest/databases/create-databases).<br></br>
    Explore all available [data sources here](/integrations/data-overview).
  </Step>
  <Step title="Create and deploy an AI/ML model">
    Create, train, and deploy AI/ML models within MindsDB via [this endpoint](/rest/models/train-model).<br></br>
    Explore all available [AI engines here](/integrations/ai-overview).
  </Step>
  <Step title="Get predictions">
    Query for predictions via [this endpoint](/rest/models/query-model).
  </Step>
</Steps>
